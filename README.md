# kiyaHomeFurniture-Deco
E-commerce JS - Shopping Cart

KiyaHomeFuriture&Deco is an online furniture shop.

On this project I've focused on the functionalities of the shopping cart. In the cart we are able to add products.
The products that we can add to the shopping cart are displayed on the page. These products were created in Contentful (contentful.com).
When I hover over the products some of them will can be added to the cart, but some are already in the cart (the text displayed on the image will be modified 
after the product has been added to the cart).
In the cart we have the selected products and the price for each item as well as the total price. We can also increase or decrease the quantity of each product, 
and at the same time remove a product or all the products from the cart.
